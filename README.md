# DriveSync 🚗

DriveSync is a full-stack car rental web application built to provide a smooth and practical vehicle booking experience.  
It includes secure authentication, booking management, and a clean responsive interface.

---

## Features

- User Registration & Login  
- Secure Authentication  
- Car Booking System  
- User Dashboard  
- Responsive UI  
- Clean Backend Structure  

---

## Tech Stack

- Node.js  
- Express.js  
- MongoDB  
- EJS  
- Tailwind CSS  
- JavaScript
